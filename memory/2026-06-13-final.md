## Known Limitations / Notes

1. **Google Maps Deprecation Warnings:** These are non-critical deprecation notices from Google Maps API. The Marker API still works perfectly.
2. **Marker Interactivity:** In some automation/testing environments, Google Maps markers may not respond to programmatic clicks due to the canvas-based rendering. They work fine in real browsers with user clicks.
3. **File Upload:** The upload button works with real image files from device camera/gallery.

## Architecture

**Frontend:**
- Pure JavaScript, no dependencies
- Google Maps API v3 (v3.55+)
- localStorage for worker name persistence
- Client-side image upload handling

**Backend:**
- PHP 8.3 on Apache 2.4
- Image upload to /bin-pics/ directory
- Airtable API integration
- Route optimization via Haversine distance calculation
- CORS configured via .htaccess

**Storage:**
- Images: /bin-pics/ on shared hosting
- Metadata: Airtable ServiceStops table
- Worker state: Browser localStorage

## 3D Cinematic Pipeline — COMPLETE ✅ (2026-07-15, Part 3)

### Architecture
Hostinger shared hosting can't run `shell_exec`/`ffmpeg`/Python → moved heavy processing to the Docker container via second cloudflared tunnel.

```
[Browser] → [agentrocketman.com UI]
                ↓ fetch tunnel URL
[Browser] → [php-tunnel.trycloudflare.com → container:9000]
                ↓ PHP built-in server
         [start-3d-video.php] → returns job_id instantly
                ↓ spawns in background
         [prepare-3d-job.php] → photos, TTS, music, subtitles
                ↓ spawns Python
         [generate_ai_walkthrough.py] → Kling clips + ffmpeg composite
                ↓ progress written to progress.json
[Browser polls] → [container:9000/check-3d-video.php]
```

### Infrastructure Running in Container
- PHP built-in server: port 9000 (serving `/public_html/`)
- Cloudflared tunnel: `bulk-exhaust-exam-prefers.trycloudflare.com` → `127.0.0.1:9000`
- Keepalive: `/data/.openclaw/workspace/php-tunnel-keepalive.sh` (restarts PHP + tunnel if either dies)

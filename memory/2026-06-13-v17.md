## User Flow (Simplified)

1. Worker sees address in **teal/green** (clickable indicator)
2. **Clicks address** (sidebar OR map tooltip)
3. **Google Maps opens** in new tab (mobile: opens native app)
4. **Full turn-by-turn with voice** starts immediately
5. Worker navigates, returns to CurbIn when done
6. **Clicks next address** → Same flow

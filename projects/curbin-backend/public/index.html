🚀 CurbIn Backend API

Service routing backend with Airtable integration and image uploads

✓ Server Status: Running

Features

  • 📸 Image Upload: POST /api/upload - Save service photos
  • 📋 Service Management: POST /api/save-service - Store service stops
  • 🗺️ Route Optimization: POST /api/optimize-route - Optimize delivery routes
  • 🔌 Airtable Sync: Automatic data synchronization
  • 💾 Local Storage: bin-pics/ folder for image uploads

Quick Start

  1. Install dependencies: npm install
  2. Set environment variables in .env
  3. Start server: npm start
  4. Test Airtable: curl http://localhost:3000/api/test-airtable

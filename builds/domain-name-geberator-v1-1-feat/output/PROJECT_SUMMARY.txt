PROJECT OVERVIEW
─────────────────────────────────────────────────────────────────────────────
An AI-powered domain name generator that creates unique, creative domain
names based on business descriptions and checks their real-time availability
using the Namecheap API.

QUICK START
─────────────────────────────────────────────────────────────────────────────
1. Upload all files to your web server
2. Navigate to: https://yourdomain.com/install.php
3. Follow the 3-step installation wizard
4. Delete install.php after installation
5. Visit: https://yourdomain.com/

REQUIREMENTS
─────────────────────────────────────────────────────────────────────────────
✓ PHP 7.4+
✓ MySQL 5.7+
✓ Apache with mod_rewrite
✓ OpenAI API key (from platform.openai.com)
✓ Namecheap API access (requires $50+ account balance)
